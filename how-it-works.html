Get your API key & create a wallet

Sign up, grab your API key, and issue a wallet to any agent in milliseconds. Each wallet has a unique agentId, owner, USDC balance, and status. You define the rules at creation time — or update them anytime.

02

Configure spending rules

Every wallet runs through NexoPay's rules engine before any transaction clears. Set daily limits, per-transaction caps, whitelisted recipients, and category locks. Add a kill switch that lets you freeze the wallet instantly from the dashboard or via API call.

Daily Spend Limit Per-TX Cap Recipient Whitelist Category Locks Instant Kill Switch
03

Agents transact autonomously

Your agent calls POST /payments to send USDC to any other NexoPay wallet. The rules engine validates in microseconds, the smart contract handles escrow if needed, and settlement lands on Base L2 — all within 2ms on average. Your agent never touches a private key.

Monitor, audit & stay in control

Every transaction is logged immutably. Your human oversight dashboard shows a real-time feed of all agent activity, spending patterns, and anomaly alerts. Export full audit logs as CSV or JSON for compliance. You're always in control — even when your agents aren't.

FAQ

Common questions.

Do my agents ever hold private keys?
Never. All cryptographic keys are managed by AWS KMS hardware security modules. Your agents interact with NexoPay via API — NexoPay handles all on-chain signing securely.
What currency do agents transact in?
USDC — a regulated, USD-pegged stablecoin issued by Circle. No price volatility, no crypto complexity. It behaves like dollars, settles like crypto.
What blockchain does NexoPay use?
Base — an Ethereum Layer 2 chain backed by Coinbase. It offers transaction fees under $0.001, sub-second finality, and full EVM compatibility. We chose it specifically for the agent use case.
Can I integrate NexoPay with LangChain, AutoGen, or CrewAI?
Yes. NexoPay is framework-agnostic. If your agent can make an HTTP request, it can use NexoPay. We ship Day 1 SDKs for Python, Node.js, and TypeScript that work natively with all major agent frameworks.
What happens if an agent goes rogue or gets hacked?
Spending rules cap the blast radius. A compromised agent can only spend up to its configured daily limit. You can freeze any wallet instantly via API or dashboard. Anomaly detection alerts you to unusual patterns in real time.
