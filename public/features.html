Features

Everything agents
need to transact.

Autonomi is built from the ground up for AI agents — not adapted from human fintech. Here's what that means in practice.

⚡
Core

Sub-2ms Settlements

Agent-to-agent transfers settle on Base L2 with on-chain finality. No batching, no waiting, no reconciliation.

  • Average 1.8ms end-to-end
  • On-chain finality via Base L2
  • Webhook confirmation on every tx
  • Transaction receipt with full metadata
🔒
Security

HSM Key Management

All private keys live in AWS KMS hardware security modules. No plaintext keys anywhere in the stack — ever.

  • AWS KMS + HashiCorp Vault
  • FIPS 140-2 Level 3 hardware
  • Zero plaintext key exposure
  • Annual penetration testing
📋
Control

Spending Rules Engine

Define per-wallet policies that enforce autonomously on every transaction before it clears.

  • Daily & per-transaction limits
  • Recipient whitelist / blacklist
  • Category-based restrictions
  • Instant kill switch via API or UI
🤝
Smart Contracts

Escrow & Task Settlement

Agents lock funds in smart contract escrow when assigning tasks. Funds release automatically on completion.

  • Trustless escrow on Base L2
  • Auto-release on task confirmation
  • Configurable timeout + auto-refund
  • Multi-party settlement support
📊
Oversight

Human Control Dashboard

Every transaction your agents make is visible in real time. You stay informed and in control at all times.

  • Real-time transaction feed
  • Anomaly & overspend alerts
  • One-click wallet freeze
  • Full audit log export (CSV/JSON)
🛠️
Developer

Best-in-Class DX

Integrate in under 30 minutes. SDKs for every major agent framework. Sandbox environment for free.

  • Python, Node.js, TypeScript SDKs
  • OpenAPI 3.0 spec + Swagger UI
  • Free sandbox with test USDC
  • LangChain, AutoGen, CrewAI support
🏦
Enterprise

Multi-Agent Treasury

Manage budgets and spending across entire fleets of agents from a single org account.

  • Org-level treasury wallet
  • Auto top-up rules per agent
  • Consolidated billing & reporting
  • Role-based dashboard access
🔔
Events

Webhook & Event System

Get notified on every payment event. Pipe Autonomi events into your own systems or agent pipelines.

  • Configurable webhook endpoints
  • Payment confirmed / failed / refunded
  • Spending limit warning events
  • Retry logic + delivery guarantees
📜
Compliance

Audit & Compliance

Immutable logs of every decision, payment, and rule change. Built for regulated industries from Day 1.

  • Tamper-proof transaction ledger
  • OFAC sanctions screening
  • KYB for enterprise accounts
  • SOC 2 Type II (in progress)
